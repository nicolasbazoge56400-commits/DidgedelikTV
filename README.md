# Documentation du Didgedelik Assistant

## Installation

Pour installer le Didgedelik Assistant, assurez-vous que vous disposez de Python 3.7 ou une version ultérieure. Vous pouvez installer les dépendances nécessaires en utilisant pip :

```bash
pip install -r requirements.txt
```

## Configuration

Avant d'utiliser Didgedelik, vous devez configurer le fichier `config.json`. Ce fichier doit contenir les informations d'authentification et les préférences de l'utilisateur. Un exemple de configuration est donné ci-dessous :

```json
{
  "api_key": "votre_cle_api",
  "paramètre_exemple": "valeur"
}
```

Assurez-vous de remplacer `votre_cle_api` par votre clé API personnelle.

## Utilisation

Une fois que tout est installé et configuré, vous pouvez lancer le Didgedelik Assistant avec la commande suivante :

```bash
python didgedelik.py
```

Suivez les instructions à l'écran pour interagir avec l'assistant.
